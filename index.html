Kotlin入門

Android開発では主にJavaかKotlinが使われる

C/C++や最近だとFlutter(Dart)も

どちらもオブジェクト指向の言語

Kotlinが言語としてモダン

  • Javaと比べ書く量が少ない
  • Nullを許容しない

Kotlinを使っていきます

Kotlinの基本情報

  • JetBrains社が2011年にリリース
  • ライセンスはApache-2.0
  • Androidはもちろん、altJSかつJVMやNativeで
    動かせるので、応用性が高い
  • 使われているとは言っていない
  • 拡張子はkt

実際に文法を見ていきましょう


								//Kotlin
								//変数の宣言と値の代入
								var 変数名: 型 = 値

								//定数の宣言と値の代入
								val 定数名: 型 = 値
							

								//具体例
								//変数の宣言と値の代入
								var a: Int = 1

								//定数の宣言と値の代入
								val a: Int = 1

開発ではvalを使うことが多い

val,varは類推型

最近の言語は変数名:型の形が多い

Kotlinは文の最後にセミコロン[;]がいらない


							//Kotlin
							//if文
							if(条件式){
								処理1
							}else{
								処理2
							}
						

							//具体例
							//if文
							if(a==0){
								println("false")
							}else{
								println("true")
							}

Kotlinでのifは文だけでなく式にもなる


							val ans = if(a==0){
								println("a is false")
							}else{
								println("a is true")
							}

最近の言語ではifを式として扱えることが多い

swichの代わりのWhen


							when(a){
								1 -> {
									println("aは1です。")
								}
								2 -> {
									println("aは2です。")
								}
								else -> {
									println("aは1でも2でもありません。")
								}
							}

こちらも式として扱える


								//Kotlin
								//for文
								for(変数名 in はじめの数..終わりの数){
									処理
								}
							

								for(i in 1..10){
									println("${i}")
								}
							

								//メソッド(関数)
								fun メソッド名(仮引数):返り値の型{
									処理
								}
							

								//具体例
								fun square(width: Int,height: Int):Int {
									return width*height
								}

クラス内にある関数をメソッドと言います

Unit型(Cで言うvoid)は返り値の型を省略可

最近は関数型プログラミングの流行っている

ラムダ式、簡単に言うと名前のない関数


							引数 -> 処理
						

							val array = arrayOf(1,2,3)
							array.forEach({num -> println(num)})

ラムダ関数に着目


							array.forEach({num -> println(num)})

最後の引数がラムダ関数の時、()を前に出せる

array.forEach(){num -> println(num)}

引数がラムダ関数のみの時、()を消せる

array.forEach{num -> println(num)}

引数が1つの時、引数をitとし処理できる

array.forEach{println(it)}

							//クラス
							class クラス名 : 継承元クラス名{
								プロパティ
								メソッド
							}
							//インスタンス化
							val 定数名 = クラス名()
							//クラス内のメソッドの実行
							定数名.メソッド名()

クラスはメソッドとプロパティ(関数と変数や定数)をひとまとめにしたもの

一般的には、以下のように使う


							class a :b{
								override fun bfun(num: Int?){
									super.bfun()
								}
							}

override: 親クラスのメソッドを上書き

super.メソッド: 親クラスのメソッドを使用
